# Proyecto React

Este proyecto fue creado con Create React App y contiene los archivos básicos para iniciar una aplicación React.

## Cómo iniciar el proyecto correctamente

1. Abre una terminal en la carpeta del proyecto:
   ```bash
   cd Proyect_React
   ```
2. Instala las dependencias del proyecto:
   ```bash
   npm install
   ```
3. Inicia la aplicación en modo desarrollo:
   ```bash
   npm start
   ```
4. Abre tu navegador en:
   ```
   http://localhost:3000
   ```

> Si aparecen errores, revisa la consola de la terminal y la consola del navegador. Asegúrate de haber instalado todas las dependencias antes de ejecutar `npm start`.

## Scripts disponibles

- `npm start`: inicia la aplicación en modo desarrollo.
- `npm test`: ejecuta las pruebas en modo interactivo.
- `npm run build`: crea una versión optimizada para producción en la carpeta `build`.
- `npm run eject`: expulsa la configuración de Create React App (operación irreversible).

## Qué aprendí en este proyecto

- Cómo usar Create React App para crear una aplicación React básica.
- Cómo ejecutar el proyecto correctamente con `npm install` y `npm start`.
- La estructura de una aplicación React básica y cómo funcionan los componentes.
- Cómo organizar componentes en `src/componentes`.
- Cómo revisar errores en la terminal y en la consola del navegador.

## Estructura principal del proyecto

- `src/App.js`: componente principal de la aplicación.
- `index.js`: punto de entrada de React.
- `src/App.css` e `index.css`: estilos principales.
- `src/componentes/Header.js`: componente del encabezado.
- `src/componentes/Menu.js`: componente del menú.
- `src/componentes/Footer.js`: componente del pie de página.
